# 査読者の概要

Aurora は、Cardano トレジャリー に対し、機関信用市場向けのオープン インフラストラクチャ (メタデータ標準、オフチェーン検証インデクサー、開発者ドキュメント、規制対象のエチオピア SACCOs を通じてインフラストラクチャを検証する現実世界のパイロット) に資金を提供するよう依頼します。

## なぜそれが重要なのか

Cardano には、eUTxO 状態、プログラム可能なローン オブジェクト、ビーコン スタイルのパターンによる検出など、ローン ファーストのクレジット市場向けのアーキテクチャ要素がすでに備わっています。欠けている層は機関の信頼インフラストラクチャです。プロの資本提供者は、基本的な融資プロトコルを許可システムに変えることなく、誰に資金提供しているのか、どのようなコンプライアンスチェックが適用されるのか、返済実績がどのように追跡されるのか、リスクがどのように報告されるのかを知る必要があります。

Aurora は、これらの懸念を核となる契約の外に置きます。 ID、検証、コンプライアンス参照、返済評判はオプションのメタデータとして添付され、オープン インデクサー インフラストラクチャによって検証されます。これにより、Pogun および将来の Cardano クレジット マーケット実装で作業を再利用できるようになります。

## コアの安全対策

- 完全な トレジャリー 割り当てに対する独立した 2-of-3 トレジャリー 受託者管理。
- 独立した署名者が多数を占める個別の 3/5 開発基金マルチ署名。
- パイロットの流動性は、30% / 30% / 40% の展開ラウンドにわたって段階的にリリースされます。
- 公開マイルストーン レポート、公開ウォレット アドレス、インデクサーの可視性、および dRep 監視ダッシュボード。
- 残りの トレジャリー 元本および トレジャリー 権利付き収益は、パイロット終了時に返還されます。
- Apache License 2.0 の下で M2 までにオープンソース リリース。
