# Aurora

**Cardano の機関信用市場向けオープンインフラストラクチャ**

Aurora は、Cardano のトレジャリーに対して、機関信用市場のための再利用可能なオープンインフラストラクチャへの資金提供を求める提案です。対象には、メタデータ標準、オフチェーン検証インデクサー、開発者向けツール、そしてエチオピアの規制対象 SACCO と実施するトレジャリー支援パイロットが含まれます。

| 項目 | 内容 |
| --- | --- |
| **トレジャリー請求額** | **2,900,000 ADA** |
| **運用予算** | **2,200,000 ADA** |
| **パイロット流動性** | **700,000 ADA**（換算時に約 USD 100,000 相当を目標） |
| **実施期間** | 12か月 |
| **主導組織** | Fairway |
| **パートナー** | Fallen Icarus、Sundial Protocol |
| **ライセンス** | Apache License 2.0 |
