# Aurora

**Cardano上の機関信用市場向けオープンインフラストラクチャ**

Auroraは、機関信用市場のための再利用可能なインフラストラクチャに資金を提供することをCardanoトレジャリーに求める提案です。対象は、オープンなメタデータ標準、オフチェーン検証インデクサー、開発者向けツール、および規制対象のエチオピアSACCOと実施するトレジャリー支援パイロットです。その目的は、基盤となる融資プロトコルをパーミッション型にすることなく、Cardanoが機関信用市場を支えられるようにすることです。

## トレジャリー資金の対象

**フェーズ1: インフラストラクチャ。** Auroraは、Cardanoネイティブ信用市場のための任意利用のメタデータおよび信頼レイヤーを作成します。本人性証明、検証ステータス、コンプライアンス参照、返済シグナルは、トランザクションメタデータを通じて融資リクエストUTxOに付与され、オープンなオフチェーン・インデクサーにより検証されます。

**フェーズ2: 実世界での検証。** インフラストラクチャは、エチオピアの貯蓄信用協同組合とのトレジャリー支援パイロットを通じて検証されます。パイロット流動性は、承認されたCardanoネイティブUSD建てステーブル資産に転換され、段階的に投入され、ローン返済に伴って再循環され、パイロット終了時にトレジャリー帰属収益とともに返還されます。

## ガバナンス上の位置づけ

Auroraは、私的な融資プラットフォームではなく、オープンインフラストラクチャとして構成されています。本提案は、開発資金とパイロット流動性を分離し、トレジャリー保管を独立受託者の下に置き、マイルストーン証拠により資金解放をゲートし、オンチェーン記録、インデクサー、公開報告、dRep監視ダッシュボードを通じて、資金投入および返済活動を独立して可視化します。
