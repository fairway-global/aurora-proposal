# 査読者向け概要

Auroraは、機関信用市場向けのオープンインフラストラクチャに資金を提供することをCardanoトレジャリーに求めます。対象は、メタデータ標準、オフチェーン検証インデクサー、開発者向けドキュメント、および規制対象のエチオピアSACCOを通じてインフラストラクチャを検証する実世界パイロットです。

## 重要性

Cardanoには、ローン起点の信用市場に必要なアーキテクチャ上の要素がすでに備わっています。すなわち、eUTxO状態、プログラム可能なローンオブジェクト、ビーコン型パターンによる発見です。欠けている層は、機関向けの信頼インフラストラクチャです。専門的な資本提供者は、誰に資金を提供しているのか、どのコンプライアンス確認が適用されるのか、返済実績がどのように追跡されるのか、また基盤となる融資プロトコルをパーミッション型にすることなくリスクがどのように報告されるのかを把握する必要があります。

Auroraは、これらの関心事項を中核コントラクトの外側に置きます。本人性、検証、コンプライアンス参照、返済レピュテーションは任意のメタデータとして付与され、オープンなインデクサーインフラストラクチャにより検証されます。これにより、本提案の成果はPogunおよび将来のCardano信用市場実装で再利用可能になります。

## 中核的な保護策

- トレジャリー配分全額について、独立した2-of-3の独立トレジャリー受託者による保管。
- 独立署名者が過半数を占める、別個の3-of-5開発資金マルチシグ。
- パイロット流動性は、30% / 30% / 40%の資金投入ラウンドを通じて段階的に解放。
- 公開マイルストーン報告、公開ウォレットアドレス、インデクサーによる可視性、dRep監視ダッシュボード。
- パイロット終了時に、残余トレジャリー元本およびトレジャリー帰属収益を返還。
- M2までにApache License 2.0の下でオープンソース公開。
